# Student Management System (Python + SQLite)

A console-based Student Management System built using Python and SQLite.  
This project manages student records, forms balanced groups, ranks students, and visualizes group performance.

---

##  Features

- Add new students
- Blacklist / Activate students
- View all students
- Rank students based on SGPA
- Automatic group formation
- Rank groups based on average SGPA
- Blacklist / Activate groups
- Graph visualization using Matplotlib

---

##  Technologies Used

- Python
- SQLite
- Matplotlib

---

##  How to Run

1. Install Python
2. Install matplotlib:
   
   pip install matplotlib

3. Run:

   python main.py
